SmartTask - README

SmartTask es una aplicación desarrollada en Java utilizando Maven. Permite la gestión eficiente de tareas, aplicando la lógica de programación, validaciones técnicas y buenas prácticas.

Composición:

Dentro del proyecto encontrarás las siguientes carpetas:

Documentación_JavaDoc -> Corresponde a la documentación JavaDoc del proyecto, para acceder dar doble clic al archivo "index.html"

Ejecutable_SmartTask -> Corresponde al archivo ejecutable en consola. Para ejecutar el programa desde la consola, se debe tener java instalado y seguir los siguientes pasos:

 	1. Abre una terminal en la carpeta raíz del proyecto (Accede a la carpeta Ejecutable_SmartTask ó a la carpeta target y en la barra de direcciones en la parte superior escribe CMD)

 	2. ejecuta el archivo .jar con el siguiente comando:
 		java -jar smarttask-1.0-SNAPSHOT.jar

Pruebas_JUnit -> Corresponde a las pruebas unitarias de la aplicación.

src, target y pom.xml -> Corresponden al código fuente del proyecto

SmartTask_Capturas_y_Coverage -> Corresponde a las capturas de pantalla del funcionamiento de la aplicación y al Coverage de la aplicación.
